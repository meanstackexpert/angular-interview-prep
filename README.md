# Angular Interview Prep

Angular interview questions grouped by difficulty level with short, direct answers.

## Sections

- Level 1 – Basics (components, modules, data binding, pipes)
- Level 2 – Routing, lifecycle hooks, services, HTTP, RxJS
- Level 3 – Change detection, performance, architecture
- Scenarios – Real-world problems and how to solve them

Each folder will contain:
- Question
- Short answer
- Code snippet (when useful)
